# 📊 Online Store Sales Analysis with Power BI

## 📌 Project Overview

This project focuses on analyzing an online store's sales performance using **Microsoft Power BI** and **DAX (Data Analysis Expressions)**. The dashboard transforms raw e-commerce data into meaningful business insights by tracking sales, profitability, targets, and regional performance through interactive visualizations.

---

## 🎯 Objective

To strengthen Power BI and DAX skills by creating calculated columns, measures, and interactive dashboards that provide valuable insights into business performance and support data-driven decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling

---

## 📂 Dataset

The project uses three related datasets:

- **List of Orders** – Customer details, order information, location, category, amount, and profit.
- **Order Details** – Category, sub-category, quantity, amount, and profit.
- **Sales Target** – Monthly sales targets for each category.

---

## 📈 Dashboard Features

- Sales Performance Analysis
- Sales vs Target Comparison
- Monthly Sales Trend Analysis
- Profit Margin Analysis
- Geographic Sales Analysis
- Profit vs Quantity Comparison
- Category-wise Sales Performance
- KPI Cards
- Matrix Reports
- Treemap Visualization
- Funnel Chart Analysis

---

## 🧮 DAX Calculations

### Calculated Columns

- Category Type
- Revenue per Order
- Sales Category

### Measures

- Total Revenue
- Order Count
- Total Target
- Average Profit in Delhi
- Year-to-Date (YTD) Sales

---

## 📊 Key Insights

- Compared actual sales against monthly targets.
- Identified high-performing product categories.
- Analyzed regional sales performance.
- Monitored monthly sales trends.
- Evaluated profitability across products.
- Measured business performance using KPIs.
- Built interactive reports for better decision-making.

---

## ✅ Conclusion

This project demonstrates the practical application of Power BI and DAX to transform e-commerce data into meaningful business insights. Through calculated columns, measures, and interactive dashboards, the analysis supports informed business decisions while showcasing strong data visualization and analytical skills.
